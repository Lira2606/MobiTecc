## Core Features:

- Delivery Form: Capture delivery information with fields for School Name, Responsible Party, Phone Number, Delivered Items, and Observations.
- Intuitive UI: A streamlined interface to create the register deliveries easily
- AI-Powered School Suggestion: Suggest frequent deliveries to a specific school, to avoid typing it every time. Uses a simple tool to find similarities.
- Offline Support: Provide offline support for recording deliveries in areas with limited connectivity. The application saves data locally, which you can upload when the connection is available again.
- Automatic notifications: Send an Email or SMS to the school once the register is made.

## Style Guidelines:

- Primary color: Light blue (#64B5F6) to convey trust and efficiency.
- Background color: Very light blue (#E3F2FD), creating a clean, uncluttered feel.
- Accent color: Soft green (#81C784) to indicate successful actions or confirmations, such as a submitted form.
- Body and headline font: 'PT Sans', a friendly and modern sans-serif, for clear readability in all form fields and labels.
- Use a consistent set of outline-style icons throughout the interface, in the primary blue color, for a professional look.
- Maintain a clean and structured layout with clear section headings to facilitate easy navigation and data entry.
